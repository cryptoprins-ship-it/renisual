# Sample photos: Houses

Drop facade photos of houses (1600px wide, JPG, < 500KB) into this folder, then
add an entry to `index.json`:

```json
[
  { "file": "vrijstaande-jaren-30.jpg", "label": "Vrijstaand jaren '30" },
  { "file": "rijtjeshuis-modern.jpg", "label": "Rijtjeshuis modern" }
]
```

The `/render` page picks them up automatically via the "Voorbeeld" tab.
